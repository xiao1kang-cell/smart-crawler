Vidaxl 数据少 · 不是 Vidaxl 限我们，是我们设了 200 上限

客户问"是我们只抓了 200 个还是对方限我们 200"——我用 curl 实测了 Vidaxl 的 sitemap_index，输入源完全公开，单站 30-50 万 SKU 都拿得到。

# 实测命令（2026-05-21 21:16 验证） curl https://www.vidaxl.de/sitemap_index.xml # HTTP 200 公开返回 # 解析结果： → 63 个子 sitemap → 其中 26 个标记为 custom-product → 第一个 product sitemap：20,000 个 SKU URL → 推算全站：26 × ~15-20K ≈ 30-50 万 SKU
📊 对比客户预期

我们当前抓 200/站 → 覆盖率 0.04-0.07%
放开 200 上限后理论上能爬到完整 30-50 万。这只是个 1 行代码的配置问题，不是技术不能。
