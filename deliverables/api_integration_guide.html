8计费

  • 按 record 计费：$1.5 per 1,000 records 返回
  • 计算口径：/api/v1/products?page_size=100 返 100 行 = 100 records
  • 免费额度：50,000 records/月（demo 期可调）
  • 查询用量：GET /api/billing/usage
  • 账单查看：Console 登录后 Settings → Billing
当前 demo 阶段：用量不收费，先跑通对接。正式上线前会另发邮件告知开始计费时间。
