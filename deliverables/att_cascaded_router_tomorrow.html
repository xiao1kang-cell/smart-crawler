1登 modem 192.168.1.254

顶部 Network → Subnets & DHCP（就是昨晚配 Public Subnet 那页）

2关掉 Public Subnet

Public Subnet 那一段，把 Public Subnet Mode 从 On 改成 Off。

这样 modem 不再尝试把 /26 段直接 ARP 给 LAN 设备（这是昨晚卡住的根因）。

3开 Cascaded Router

页面最底下 Cascaded Router 段，填：

字段值
Cascaded Router EnableOn
Cascaded Router Address192.168.1.91（iMac 的私网 IP）
Network Address108.95.61.128
Subnet Mask255.255.255.192

点 Save。modem 不一定重启，可能立刻生效。

原理：modem 收到 dst=108.95.61.X 的包，直接转给 192.168.1.91（iMac），不再要求 ARP 学 /26 内每个 IP。iMac 自己接管整个段。
