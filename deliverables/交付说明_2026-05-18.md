### 评论平台
- **Trustpilot 9 个国别站**：~1,346 条评论（含 SKU/评分/原文/语言/商家回复）

- **TrustedShop (DE) / Opiniones Verificadas (ES, PT)**：走平台后台 CSV/Excel 导出 + 我方 `cli reviews-import` 入库（feed 路径）

### Google Shopping
- 关键词管理 / 入库 / 竞品占有率分析 / 排名趋势 / 新竞争者识别 全部就绪
- 采集器（patchright + selectolax）就绪；Google 对消费级 IP 启用 reCAPTCHA，**下一步对接 SerpApi 商用 SERP**

### Agent 接入（行业首发）
- **MCP 服务器**：`smartcrawler.io/mcp`（streamable-http），9 个工具

- **REST API**：`/api/v1/products` / `/api/v1/promotions` / `/api/v1/reviews` / `/api/v1/shopping/*` / `/api/v1/trends` / `/api/v1/site/{site}` 等
- **发现层**：`/llms.txt`、`/agents.json`、`/.well-known/mcp.json`、`/.well-known/ai-plugin.json`

## 七、技术亮点

1. **Agent-First 设计**：MCP + REST 双接入，行业首个把竞品采集做成 AI Agent 工具的产品
2. **平台自适应**：shopify / nuxt / vue_spa / magento / vidaxl / generic 6 套采集器自动按 `site.platform` 路由
3. **反封锁基建**：熔断 + 限速档 + 站点冷却 + IP 配额 + 浏览器指纹（`app/antiban.py`）
4. **数据回溯**：每次采集落 `data/snapshots/<site>/<date>/<slug>.gz`，可重放
5. **可扩展性**：新增标杆站点只需 `sites.yaml` 加一行，`_seed_sites` 自动同步入库
