Market Direction · CIA Summary

smart-crawler 不该卖“通用爬虫”，而该卖“电商竞品异动情报”。

底层能力可以通用，但商业定位要垂直。真正可累积的壁垒不是今天抓到一个页面， 而是长期、连续、归一化的商品、价格、促销、评论和类目动作数据。

推荐路线：跨境电商品牌的竞品新品、类目扩张、价格促销、VOC 痛点和热度趋势雷达。
短期做每日异动，中期做类目机会雷达，长期做跨境电商商品情报数据库。

一句话定位

竞品异动情报系统 每天追踪上新、价格、促销、评论、热度趋势。
不是库存监控 库存只能做辅助信号，单独价值不够高。
不是精准销量承诺 销量更适合做热度信号、动能评分和置信度分层。

CIA 判断

用 Customer、Industry、Alternatives/Competitors 三个维度过滤后，最好的方向不是更泛，而是更像一个行业数据资产。

C · Customer

买单客户是品牌方和类目负责人

他们要的不是“页面内容”，而是竞品最近在押什么、价格带怎么变、差评暴露了什么机会。

  • 跨境 DTC / 独立站品牌
  • 家居、户外、家具、工具类目
  • 运营、产品经理、类目负责人、老板
I · Industry

单点工具很多，统一情报层稀缺

价格监控、评论管理、Amazon 工具都成熟，但跨独立站、marketplace、评论平台、Google Shopping 的归一化情报仍有空间。

  • 价格监控是刚需但红海
  • VOC 市场在增长但多聚焦自家评论
  • 类目扩张和新品动作更像战略信号
A · Alternatives

替代品强在点，弱在组合

Helium 10、Jungle Scout 强在 Amazon；Prisync 强在价格；Birdeye 强在评论；Apify/Bright Data 强在基础设施。

  • 空位：跨平台商品情报
  • 空位：竞品评论转产品机会
  • 空位：长期历史数据形成指数

功能优先级

库存保留，但不作为主卖点。销量预估保留，但包装成热度趋势。

Rank 方向 为什么值得做 产品表达
01 竞品新品与类目扩张 比单 SKU 降价更接近战略动作，能回答“竞品在押哪个市场”。 Category Opportunity Radar
02 价格 / 促销异动 数据硬、可验证、客户容易理解，是每日监控的刚需入口。 Daily Price & Promo Watch
03 竞品差评 / VOC 机会 不要做评论回复工具，要做产品缺陷、卖点机会和风险预警。 VOC Opportunity Radar
04 热度趋势 / 销售动能 声量高，但不能承诺精准销量；适合做排序、趋势和置信度。 Momentum Score
05 库存 / 上下架变化 单独价值弱，和促销、评论、热度一起看才有意义。 辅助信号，不做主标题

数据量足够大之后的机会

每天实时抓取只是入口。真正的商业升级来自长期历史、商品归一和可解释模型。

Opportunity 1

类目机会发现

识别哪些类目上新加速、价格带下移、促销强度上升、差评集中但热度仍高。

Opportunity 2

商品匹配与竞品图谱

把同款、近似款、替代款、同价格带款连接起来，回答“我的产品该对标谁”。

Opportunity 3

市场指数

沉淀类目价格指数、促销强度指数、新品活跃度指数、差评风险指数、品牌扩张指数。

Opportunity 4

客户自有数据对标

接入客户价格、销量、评论后，输出价格分位、评分差距、促销过度、listing 改进建议。

路线图

按价值密度推进：先做客户每天会看的东西，再做别人补不回来的数据壁垒。

0-2 个月

每日竞品异动

新品、降价、促销、评论、热度趋势。目标是让客户每天打开，或者每天收到一封可行动报告。

  • Daily delta 报告
  • 告警规则
  • Excel / API / MCP 交付
2-6 个月

类目机会雷达

从 SKU 监控升级到类目判断，输出“哪个国家、哪个价格带、哪个产品形态正在变化”。

  • 类目归一
  • 新品活跃度
  • VOC 机会聚类
6-18 个月

电商商品情报数据库

形成跨品牌、跨国家、跨平台、跨时间的商品知识图谱和指数产品。

  • 商品匹配图谱
  • 市场指数 API
  • 客户经营对标

最终判断

smart-crawler 的长期机会不是库存，不是单点销量预估，也不是通用爬虫。 最值得押的是：基于长期抓取数据的类目级机会发现、商品级竞品图谱和 VOC 洞察。

做数据资产，不做爬虫工具
参考方向： Helium 10 / Jungle Scout 代表 Amazon seller intelligence； Prisync / SoftwareReviews / Centric 代表价格情报； Birdeye / Reputation / Yotpo 代表评论与声誉管理； Apify / Bright Data / Firecrawl 代表通用爬虫和数据基础设施。 本页结论基于 smart-crawler 当前项目资产与 CIA 框架归纳。
