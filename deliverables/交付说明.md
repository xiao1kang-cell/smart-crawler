# smart-crawler 交付说明

> 交付日期：2026-05-16　|　范围：遨森标杆数据采集平台 MVP（P0 三品牌）

## 一、这次交付了什么

把《需求规格说明书》里的「输入 = Aosom 标杆网站、输出 = 数据交付物」**真正跑起来了** ——
一个能采集、能清洗、能展示、能导出的全栈系统，不是 PPT，是可运行代码 + 真实数据。

## 二、实测结果（2026-05-16 首次采集）

| 标杆站点 | 平台类型 | 采集到的数据 | 促销识别 |
|----------|----------|--------------|----------|
| SONGMICS US | Shopify | 4,153 SKU / 1,494 SPU / 449 个分类 | 580 条 |
| Homary US | Nuxt SSR | 演示 150 条（站内共 4,347 商品） | — |
| Costway US | Vue3 SPA | 1,056 SKU（评分/库存/促销全有） | 925 条 |

三个 P0 品牌全部跑通。三类技术栈（Shopify / Nuxt / Vue SPA）各用一套采集方案：

- **SONGMICS** —— 标准 Shopify，`/products.json` 直拉，最快最稳
- **Homary** —— Nuxt SSR，解析 sitemap + 服务端渲染 HTML
- **Costway** —— Vue SPA，`/api/*` JSON 接口 curl_cffi 直连（实测无需住宅代理）

## 四、怎么跑起来

```bash
cd ~/smart-crawler
./run.sh              # 一键：建环境 + 装依赖 + 启服务
# 浏览器打开 http://localhost:8000
```

看板里点「采集当前站点」即可触发实时采集；点「数据导出」下载 Excel。
