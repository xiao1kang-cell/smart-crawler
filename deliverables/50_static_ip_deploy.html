50 静态 IP × smart-crawler · 5 分钟接入

每台机器跑 1 个 docker，开 HTTP proxy 端口 8087 → 给我 50 行 URL → 我加到代理池 → 立即解锁 Wayfair / eBay / Walmart / Home Depot 等所有 PerimeterX/Akamai 反爬。

1方案选择

基于你的条件（50 US 静态 IP · 公司局域网后台可控）

方案优势劣势
HTTP Proxy（推荐） · tinyproxy / 3proxy单文件 / 单端口 / 资源低（<5MB RAM）· 加 Auth 防滥用HTTP（非 SOCKS5）· 99% 站点足够
SOCKS5 · dante-serverUDP 支持 / 更隐蔽配置稍复杂
SSH tunnel0 安装（仅需 ssh）每台得给 SSH key
推荐 HTTP Proxy：50 台机器跑同一 docker 命令，1 分钟一台，统一端口 8087 + 统一密码。

2每台机器跑这一行

SSH 进每台机器，跑一次。开机自启。

Linux / Mac / Synology / 树莓派 都通用

# 方案 A · 一行 docker 命令（推荐）
docker run -d --name sc-proxy --restart unless-stopped \
  --network host \
  -e BASIC_AUTH="proxy_user:proxy_password" \
  -e PORT=8087 \
  shenoise/tinyproxy:latest

或者 docker-compose 版本

# docker-compose.yml on each device
services:
  sc-proxy:
    image: shenoise/tinyproxy:latest
    restart: unless-stopped
    network_mode: host
    environment:
      - BASIC_AUTH=proxy_user:proxy_password
      - PORT=8087
      - ALLOWED_NETWORKS=0.0.0.0/0

路由器后台需要做的

  • 每台设备的 静态 IP 已配 ✓（你说有了）
  • 路由器/防火墙开 TCP 8087 端口的 inbound（仅限 smart-crawler NAS IP 来源更安全）
  • NAS 出口 IP：smartcrawler.io 对应的 Cloudflare egress 或你的 NAS 公网 IP（告诉我也可以加白名单）
安全：BASIC_AUTH 用户密码请改，不要用 demo 的 proxy_user:proxy_password。也可以加 IP 白名单只让 NAS 访问。

6批量部署脚本（如果你想自己跑 50 台）

#!/bin/bash
# 在 50 台机器上同时部署。需准备 hosts.txt（每行一台机器 IP）

USER="smart"
PASS="YOUR_STRONG_PASSWORD"
PORT=8087

for host in $(cat hosts.txt); do
    echo "Deploying to $host..."
    ssh root@$host "docker run -d --name sc-proxy --restart unless-stopped \
        --network host \
        -e BASIC_AUTH='$USER:$PASS' \
        -e PORT=$PORT \
        shenoise/tinyproxy:latest"
done

# 验证全部 50 台
for host in $(cat hosts.txt); do
    ip=$(curl -s -x http://$USER:$PASS@$host:$PORT https://api.ipify.org --max-time 5)
    echo "$host -> $ip"
done
