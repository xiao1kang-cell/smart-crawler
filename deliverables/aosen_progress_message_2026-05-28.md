**回应 5/27 反馈**："数据太少，vidaxl.nl 一个 furniture 大类就有 150k 产品"

## 已修复的 4 个工程根因

1. **应抓数据校准**：之前 dashboard 显示 vidaxl_nl 应抓 6,000（自我设限），实际 sitemap 是 **304,024 URL**。已改为读真实 sitemap 总量。
2. **Resume 推进**：之前每次 run 都从 sitemap 头部抓，多轮重复同一批 URL。已加 product_url 去重，让每轮真实推进。
3. **并发 / 反爬限流**：每个 run 从 6 小时单线程 → 5-8 分钟 10 并发。加 random.shuffle 提高 unique parent 命中率。
4. **Worker 死亡 bug**：signal.alarm 在子线程引发 ValueError 致 worker 静默死亡，已修复。
