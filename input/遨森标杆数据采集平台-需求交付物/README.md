## 实地验证结果（P0站点）

| 站点 | 技术栈 | 反爬难度 | 结论 |
|------|--------|----------|------|
| SONGMICS | Shopify | 低 | JSON API 全开放，最容易采集 |
| Costway | Vue3 SPA | 极高 | ASN级封锁，必须美国住宅代理+Playwright |
| Homary | Nuxt.js | 中 | SSR预渲染可HTTP抓基础数据 |
| Trustpilot | Next.js | 高 | AWS WAF质询，需stealth浏览器 |
