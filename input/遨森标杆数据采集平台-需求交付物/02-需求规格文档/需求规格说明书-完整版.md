# 遨森标杆数据采集与分析平台 — 需求规格说明书

> 版本: v1.0 | 日期: 2026-05-16 | 状态: 待评审

---

## 1. 项目背景与目标

### 1.1 背景

遨森（Aosom）是跨境电商家居品牌，业务覆盖 US/UK/DE/FR/IT/ES/IE/RO/PT/NL/PL/CA 共 12 个站点。为提升市场竞争力，需要系统性地监控标杆竞品网站的业务信息、产品信息、营销策略及消费者口碑数据，以数据驱动营销和产品决策。

现有状态：仅有手工一次性导出的 SONGMICS US 站点样本数据（905 个 SKU），无持续运行的采集系统。

### 1.2 项目目标

| 目标类型 | 描述 |
|----------|------|
| 功能目标 | 建立系统，定期采集标杆网站和第三方评论平台数据，完成清洗、分析和展示 |
| 质量目标 | 数据采集准确率 ≥ 95%，字段完整率 ≥ 90% |
| 扩展性目标 | 新增标杆站点可在 1-2 天内完成配置上线 |
| 成本目标 | 分层采集策略，按性价比优化，控制持续运营成本 |

### 1.3 预期收益

- 增强对竞品动态和市场趋势的实时监控能力
- 了解消费者对 Aosom 及竞品的反馈，改进产品和服务
- 数据驱动决策，提高营销和产品策略精准性
- 降低人工数据采集和分析的人力成本

## 4. 功能需求 — 结构化清单

### 4.1 模块一：标杆网站数据采集与分析（P0）

#### 4.1.1 业务概览数据采集

| 编号 | 需求 | 优先级 | 详述 |
|------|------|--------|------|
| F1-001 | SPU 数量统计 | P0 | 采集站点全部商品，统计去重后的 SPU（Standard Product Unit）总数 |
| F1-002 | SKU 数量统计 | P0 | 采集站点全部 SKU（含变体），统计总数及按品类分布 |
| F1-003 | 品类数统计 | P0 | 解析站点分类导航树，统计一级/二级/三级类目数量 |
| F1-004 | 销量估算 | P0 | 通过评论数量变化倒推销量（核心业务逻辑，见下方说明） |
| F1-005 | 营收估算 | P0 | 销量估算 × 商品售价 = 预估营收 |
| F1-006 | 趋势数据汇总 | P0 | 按日/周/月维度汇总上述指标，计算同比/环比 |

**销量估算逻辑说明**：
- 评论数量变化 = 本期评论数 - 上期评论数
- 销量 ≈ 评论增量 / 评论率（评论率需按品类设定经验系数，通常 1%-5%）
- 此为估算值，需在报表中明确标注为"预估"

**新品识别规则**（需明确）：
1. 首次出现规则：在我方数据库中首次出现的 SKU 标记为新品，3 个月后取消新品标签
2. 站点新品页规则：若标杆网站有专门的 "New Arrivals" 页面，直接采集该页面的商品
3. 优先使用规则 2（更真实），规则 1 作为补充
4. 需注意：3 个月后再上线的 SKU 可能是重新上架而非新品

**热销品识别规则**（多维度，按优先级）：
1. 网站自身标签：商品页面标注 "Best Seller" / "Top Rated" / "Popular" 等标签
2. 评论增量排名：按评论数量变化倒推的销量排名
3. 排序排名：利用站点 "Sort by Best Selling" 功能时的排序位置

**活动识别逻辑**：
- 主要方式：对比同一 SKU 连续两次采集的价格变化，价格下降时判定为活动
- 辅助方式：解析商品详情页展示的活动标签/横幅/倒计时
- 活动结束判定：价格恢复到活动前水平
- 需支持识别的活动类型：直降、满减、买赠、免运费、限时闪购

| review_topics | string[] | N | 平台自动归类的话题标签 | Trustpilot 有 17 种（delivery_service/product/quality 等） |
| collected_time | datetime | Y | 采集时间 | 系统生成 |

**Trustpilot 评论字段可用性说明**：
- Trustpilot 评论为**商家维度**，不绑定具体 SKU/商品 URL（原需求文档说"有 SKU 字段"仅指 Trustpilot Business 后台导出时有关联）
- Trustpilot 页面内嵌 JSON-LD 包含完整评论数据（author、datePublished、headline、reviewBody、reviewRating），突破 AWS WAF 质询后可直接解析
- Trustpilot 自带话题标签（review_topics）可用于预分类，减少 NLP 分析工作量

**差评定义**：评分 ≤ 3 星的评论为差评

## 15. 站点技术档案（实地爬取验证 2026-05-16）

> 以下为 P0 站点的实地爬取结果，为开发团队提供直接可用的技术情报。

### 15.1 SONGMICS（致欧）— Shopify 平台

| 维度 | 详情 |
|------|------|
| 平台 | Shopify（主题 Gecko v5.8） |
| 内部域名 | zielhome-us.myshopify.com |
| CDN/WAF | Cloudflare（未启用 Bot 挑战） |
| JS 渲染 | **不需要**（JSON API 全开放） |
| 反爬难度 | **低** |
| 商品总数 | 1,496 SKU（sitemap 确认） |
| 分类数 | 449 个 collection |

**推荐采集方案**：
```
# 全站商品（无需浏览器，纯 HTTP）
GET /products.json?limit=250&page={N}      → 全量 SKU 数据
GET /collections/{handle}.json             → 分类信息
GET /collections/{handle}/products.json    → 分类下商品

# 特殊页面
/collections/new                           → 新品（71 个）
/collections/top-picks                     → 热销品（18 个）
```

**可用的 Shopify JSON API 字段**：id, title, body_html, vendor, product_type, handle, tags[], variants[](id/sku/price/compare_at_price/option1-3/available/inventory_quantity/grams), options[], images[](src/width/height), published_at, created_at, updated_at

**JSON-LD 额外字段**：mpn, gtin13, aggregateRating(ratingValue/reviewCount), shippingRate, deliveryTime, merchantReturnPolicy

**评论系统**：Ryviu（Web Component 动态加载），需 JS 渲染或 Ryviu API 才能采集评论内容

### 15.2 Costway（豪雅）— Vue 3 SPA

| 维度 | 详情 |
|------|------|
| 平台 | 自建 Vue.js 3 SPA + Ant Design Vue |
| CDN/WAF | Cloudflare（**ASN 级别封锁**，国内 IP 直接 403） |
| JS 渲染 | **强制需要**（纯 SPA，HTML 只有空壳） |
| 反爬难度 | **极高** |
| 风控系统 | Forter（行为分析）+ Emarsys（追踪） |

**推荐采集方案**：
```
# 必须：美国住宅代理 IP + Playwright
# API 端点（需先获取有效 Session）：
GET /api/products?category_id={id}&page={N}&pagesize=48
GET /api/product/{product_id}?piid={variant_id}
GET /api/customer/get_product_review_list?product_id={id}
GET /api/home-bestseller                    → 畅销榜
GET /api/home-newarrivals                   → 新品
GET /api/spike_list                         → 限时抢购
```

**独有字段**：plus_price（会员价）、pts（积分）、ar_model（AR 模型）、spkie_special_time（限时降价倒计时）、is_pickup（门店自提）

### 15.3 Homary（爆米科技）— Nuxt.js

| 维度 | 详情 |
|------|------|
| 平台 | Nuxt.js（Vue.js SSR） |
| CDN/WAF | Cloudflare（图片层） |
| JS 渲染 | **部分需要**（基础数据 SSR 预渲染，评论/变体需 JS） |
| 反爬难度 | **中** |

**推荐采集方案**：
```
# 分类列表（SSR，HTTP 可抓）
GET /{category}/f0/{subcategory}-{id}-p{N}.html

# 商品详情（JSON-LD 含基础数据，评论需 JS）
GET /item/{slug}-{spu_id}.html

# Sitemap
GET /sitemaps/google_sitemap_main_us.xml    → 9 个子文件
```

**独有字段**：sub_title（副标题）、act_info（活动详情含价格/时间/库存）、weight（重量）、load_capacity（承重）、package_info（包装箱数量及尺寸）、installation_guide_url（安装指南 PDF）

### 15.4 Trustpilot — Next.js + AWS WAF

| 维度 | 详情 |
|------|------|
| 平台 | Next.js SSR |
| WAF | AWS WAF + reCAPTCHA |
| JS 渲染 | **需要**（首次访问需通过 WAF 质询） |
| 反爬难度 | **高** |
| Aosom 评论总数 | 1,886 条（US 站） |
| 商家回复率 | 97% |
| 每页评论数 | 20 条，约 95 页 |

**推荐采集方案**：
```
# Playwright + stealth 插件突破 WAF
GET /review/www.aosom.com?page={N}

# 页面内嵌 JSON-LD 包含完整评论数据（突破 WAF 后直接解析）
# 话题标签 API（公开）
GET /api/businessunitprofile/businessunit/57d43f540000ff0005948a3b/service-reviews/topics
```

**Trustpilot 话题标签（17 种）**：delivery_service, product, order, price, quality, customer_service, instructions, service, location, customer_communications, user_experience, mistake, value_for_money, website, solution, refund, warranty

### 15.5 各站点反爬分层代理策略建议

| 站点类型 | 代理方案 | 预估月成本 |
